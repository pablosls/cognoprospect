# Cogno Prospect

Projeto Desenvolvido para o Mega Hack 2020. 

Tecnologias utilizadas 
 
 - NodeJS
 - MySQL
 - Python (Crawler)
 - HTML5/CSS

Instalando o projeto 

1) git clone 
2) criar um banco de dados MySQL
3) Rodar Script para criação das tabelas
3) Configurar arquivo de conexão
4) npm install 
5) npm start
